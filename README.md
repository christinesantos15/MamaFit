MamaFit 💙

MamaFit is a custom Progressive Web App (PWA) designed to simplify and track home workouts based on Coach's Program.
It was built to provide a seamless, mobile-first experience that works even without an internet connection.

🌟 Key Features
- Structured Routines: Integrated Day 1 (Lower Body), Day 2 (Upper Body), and Day 3 (Arms,Abs,Glutes) schedules.
- Interactive Counter: A logic-based rep and set counter specifically for lifiting days (10 reps per set, 2-3 sets per exercise.
- Daily Cardio Tracking: Quick access to walking and step goals (Starter, Mid, and Advance levels).
- Persistence: A streal system that saves progress locally, encouraging consistency.
- PWA Ready: Can be installed directly onto a mobile home screen via the browser for a native app feel/.

🛠️ Tech Stack
- Frontend: HTML5, CCS3, JavaScript(Vanilla)
- Data Persistence: localStorage API
- PWA Features: Web App Manifest and Service Workers for offline capabilities.

🏃 How to Use
1. Choose the workout day assigned by the coach.
2. Use the +1 Rep button to track your current set; the app will automatically progress through the required sets (2 for warm-ups, 3 for main lifts).
3. Complete the session to update your daily streak.
